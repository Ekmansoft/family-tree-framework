# Family Tree Framework

## Overview
The Family Tree Framework is a modern web application designed to import, visualize, and edit genealogy family trees from GEDCOM files. It provides a user-friendly graphical interface that allows users to navigate through family relationships, edit individual details, and manage connections between family members.

## Features
- **GEDCOM Import**: Easily import family trees from GEDCOM files.
- **Graphical Interface**: Intuitive UI for visualizing family trees.
- **Editing Capabilities**: Edit individual person details and relationships.
- **Undo/Redo Functionality**: Manage changes with built-in undo and redo support.
- **Customizable Toolbar**: Access common actions like saving and undoing changes.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node package manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/family-tree-framework.git
   ```
2. Navigate to the project directory:
   ```
   cd family-tree-framework
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Demo
To see the framework in action, you can run the demo:
1. Navigate to the demo directory:
   ```
   cd examples/demo
   ```
2. Open `index.html` in your web browser.

### Building the Project
To build the project for production, run:
```
npm run build
```
